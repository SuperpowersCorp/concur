Concur

Concur is a brand new Web UI Framework for Haskell that seeks to take the best parts from Elm, Reflex, and React.

More docs will be forthcoming. Until then check out the github repo for more information.

Also check out the examples below -

Concur • Examples

  1. Click Counting Example - Demo, Source - Count total number of clicks on the page, with a button that increments the click count by 10, and also autoincrement clicks every second.
  2. TodoMVC Example - Demo, Source - The canonical TodoMVC example, with views modeled after the one in Elm.
  3. Mario Example - Demo, Source - Port of the Mario Example from Elm
